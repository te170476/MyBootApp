STS(Spring Tool Suite) 入門

## 文字コードはUTF-8に統一
STS -> [Windows] -> [Preferences]
において、
- [General] -> [Workspace]
- *Text file encoding* 欄にて、
  Other(UTF-8) を選択

- [Web] -> [JSP Files]
- Encoding 欄にて、
  *ISO 10646/Unicode(UTF-8)* を選択
